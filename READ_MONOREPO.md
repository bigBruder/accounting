# Monorepo: Accounting & Camp Manager

Цей проєкт було перетворено на **монорепозиторій**, що дозволяє керувати декількома застосунками одночасно та перевикористовувати спільний код (компоненти, стилі).

## Структура проєктів
- `apps/family-budget` — Основний застосунок для сімейного бюджету.
- `apps/camp-manager` — Новий функціонал для облік витрат на табір.
- `packages/shared-ui` — Спільні компоненти та стилі (Premium UI).

## Як запустити
Для початку встановіть залежності в кореневій папці:
```bash
npm install
```

### Запуск Сімейного Бюджету:
```bash
npm run dev:family
```

### Запуск Табірного Обліку:
```bash
npm run dev:camp
```

## Функціонал Табірного Обліку
1. **Збір коштів**: Додавайте внески від батьків або спонсорів.
2. **Витрати**: Фіксуйте кожну покупку з категоріями.
3. **Баланс**: Автоматичний підрахунок "Плюс-Мінус" та фінальна сума залежно від фільтрів.
4. **Експорт**: Кнопка "Експорт CSV" дозволяє завантажити таблицю для звітності.
